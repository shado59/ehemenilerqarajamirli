# Qaracəmirli Əhəmənilər Sarayı — Next.js skelet

Layihə skeleton-u: Next.js (App Router) + TypeScript + Tailwind CSS. Bütün mətn və şəkillər `src/data/sarayData.json`-dən yüklənir.

Quraşdırma:

```bash
npm install
npm run dev
```

Qeyd: Sonradan real şəkilləri `public/` qovluğuna yerləşdirin və `src/data/sarayData.json`-dəki `img`/`image` yollarını uyğunlaşdırın.
